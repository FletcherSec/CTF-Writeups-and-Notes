- Basic Support - free
	- Customer Service and Communities - 24/7 customer service, documentation, whitepapers, and support forums
	- AWS Trusted Advisor - Access to the 7 core [[Trusted Advisor]] checks and guidance to provision your resources following best practices to increase performance and improve security
	- AWS [[Health Dashboard]] - a personalized view of the health of AWS services, and alerts when your resources are impacted

- AWS Developer Support Plan
	- Business hours email access to Cloud Support Associates
	- Unlimited cases / unlimited contacts
	- Case severity / response times:
	- General guidance < 24 hours and System impaired < 12 hours

- AWS Business Support Plan (24/7)
	- Intended to be used if you have production workloads
	- [[Trusted Advisor]] - Full set of checks + API access
	- 24/7 phone, email, and chat access to Cloud Support Engineers
	- Unlimited cases / contacts
	- Access to Infrastructure Event Management for additional fee
	- Case severity / response times: General guidance < 24 hours and System impaired < 12 hours and production system impaired < 4 hours and production system down < 1 hour

- AWS Enterprise On-Ramp Support Plan (24/7)
	- Intended to be used if you have production or business critical workloads
	- Access to a pool of [[TAM]] (Technical Account Managers)
	- Concierge Support Team (for billing and account best practices)
	- Infrastructure Event Management, Well-Architected and Operations Reviews
	- Case severity / response times: General guidance < 24 hours and System impaired < 12 hours and production system impaired < 4 hours and production system down < 1 hour and business critical down < 30 minutes
	
- AWS Enterprise Support Plan (24/7)
	- Intended to be used if you have mission critical workloads
	- Access to a designated [[TAM]]
	- Concierge Support Team (for billing and account best practices)
	- Infrastructure Event Management, Well-Architected and Operations Reviews
	- Access to AWS Incident Detection and Response (for an additional fee)
	- Case severity / response times: General guidance < 24 hours and System impaired < 12 hours and production system impaired < 4 hours and production system down < 1 hour and business critical down < 15 minutes
