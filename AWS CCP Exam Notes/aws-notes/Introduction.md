Servers composed of:
- CPU
- RAM (memory)
- Storage -> [[Database]]
- Networking: Routers, switches, DNS, etc.

Cloud is an externalization of home networking "on demand"

Cloud has:
- Pay as you go pricing
- Provision exact type and size of computing you need
- Access almost instant
- Simple UI to access servers, storage, etc.

#### Cloud Characteristics:
- On-demand self service
- Broad network access
- Multi tenancy and resource pooling
- Rapid elasticity and scalability
- Measured service (users pay for exactly what they use)

##### Advantages of Cloud
Trade capital expense [[CAPEX]] for operational expense [[OPEX]]
- Pay on demand: don't own hardware
- Reduced total cost of ownership [[TCO]] and Operational Expense (OPEX)
- Benefit from massive economies of scale
- Stop guessing capacity
- Increase speed and agility
- Don't spend money managing data centers
